Technical Analysis
------------------

1) 指令/访存类型敏感度
- 访存密集测试均值: 6.63x；非访存密集测试均值: 6.41x。
- 原因：`--cache-penalty 10` 会同时放大取指与数据访问代价，但数据访问可被 D-Cache 局部性显著吸收，因此 load/store 类测试受益更大。

2) 命中率与加速比关系
- 全量样本中，Speedup 与 D-Cache 命中率相关系数约为 0.530，与 I-Cache 命中率相关系数约为 0.703。
- 本轮数据中 I-Cache 相关性更高，说明取指路径与控制流结构在该测试集里对 speedup 影响更突出。
